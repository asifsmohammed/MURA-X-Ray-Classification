How to run:
1. Open the above link to accept the project folder in to your drive
2. From the shared with me of your Drive right click the MURAv1.1 folder and click 'add shortcut to drive'
3. Open Project.ipynb file from the created shortcut
4. Mount the Google Drive to the notebook from the tool bar on the left hand side
5. (Optional) If there are any errors  in the path, then please make sure to check whether the folder is accessible from the above path
	using the following command -> os.lisdir()
6. Run every cell in the order on which it is written
7. Done
